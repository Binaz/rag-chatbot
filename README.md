# Energy & Sustainability RAG Chatbot

**A Retrieval-Augmented Generation (RAG) chatbot** that answers questions using research papers related to **Energy and Sustainability**.  
This project demonstrates how modern NLP and embedding-based retrieval can turn unstructured research PDFs into an interactive, question-answering assistant.

## Project Overview

The RAG Chatbot combines:
- **Document retrieval** using semantic search (FAISS + Sentence Transformers)
- **Context-aware text generation** using an LLM (`microsoft/phi-2`)
- **Gradio interface** for interactive querying

It was developed using **Google Colab**, **Hugging Face models**, and **FAISS** for vector similarity search.

---

## Architecture

PDFs → Text Extraction → Chunking → Embeddings → FAISS Index → RAG Pipeline → Gradio Chatbot

---

**Key Components:**
1. **PDF Extraction:** `PyMuPDF` extracts text from Energy & Sustainability papers.
2. **Chunking:** Text is divided into manageable segments (~500 tokens).
3. **Embeddings:** `sentence-transformers/all-MiniLM-L6-v2` encodes chunks.
4. **Retrieval:** FAISS retrieves top-K relevant chunks for a query.
5. **Generation:** `microsoft/phi-2` generates context-aware answers.
6. **Interface:** Gradio provides an easy web interface for users.

---

## Features

- Answers domain-specific questions about Energy & Sustainability  
- Uses semantic search for context retrieval  
- Lightweight LLM for free, GPU-friendly inference  
- Modular code design (`src/` directory)  
- Interactive Gradio chat interface  
- Fully reproducible on Google Colab

## Data Storage

Due to storage and licensing constraints, full research PDFs are not included.

To reproduce:
1. Create a folder data/pdfs/
2. Download 20-30 Energy & Sustainability papers from open-access sources such as:
   - arXiv
   - SpringerOpen
   - ScienceDirect Open Access
3. Add them as .pdf files. Example:
```bash
data/pdfs/
├── energy_storage_methods.pdf
├── renewable_energy_policy.pdf
├── pfas_remediation_techniques.pdf
```
